# Changelog

## [1.3.0] - 2026-01-09

### Added
- **高级弹幕设置**：支持自定义弹幕透明度、字体大小、解析速度和显示区域（全屏/半屏等）。
- **个性化主题**：新增主题配色选择功能（粉色、蓝色、紫色、绿色）。
- **画质适配**：支持设置弹幕渲染画质（720P/1080P/2K/4K）。
- **链接交互**：点击作者信息可复制 GitHub 项目地址。
- **任务完成弹窗**：合并完成后显示统计信息及失败项目列表。
- **重置功能**：一键恢复默认设置。

### Internal
- 引入 `url_launcher` 和 `flutter_launcher_icons` 扩展功能。
- 优化了项目目录结构，清理了大量调试日志。
